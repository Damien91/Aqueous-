# Aqueous-


### Build

#### Requirements
- You need to have NodeJS > 8.0
- Install TypeScript and TSLint
```console
npm install -g typescript tslint
```
- Install the dependencies of this project.
```console
npm install
```
- Run the first build
```console
npm run bulid
```

#### Development
- To test
```console
npm run test
```
- To build
```console
npm run build
```
- To run dev server
```console
npm run start
```
